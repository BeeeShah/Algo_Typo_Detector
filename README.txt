This progarm will will accept a target string and a typo string, and it will compute the minimum cost
for transforming the target string into the typo string, as well as a sequence of typos with this cost. 
This solution is using dynamic programming.

In order to run the code make sure all the files are in the same directory.

Files: input.txt, new.cpp

The program will display the answer as well as create and print to output.txt

To compile the program type: "g++ -o new *.cpp"
To run the program type: "./new"
